// Key Features

選ばれる3つの理由

「こんなツールがほしかった」を形にしました。

01 / 03
⚡
集計時間ゼロへ 面倒な手作業を丸ごとお任せ

「各サイトにログインして、CSVをダウンロードして、スプレッドシートに貼り付ける」— その繰り返しをすべてパソコンが代わりにやってくれます。 自動取得設定をONにすれば、毎朝起動すら不要。完全放置でOKです。

全サイト一括 スケジュール自動化 CSV不要
02 / 03
📊
即データ活用 そのまま分析・給与計算に

取得データはGoogleスプレッドシートに自動で整理して記録。 日付・サイト・キャスト名・ポイントがそろった状態で蓄積されるので、 グラフ作成・給与計算・レポートにすぐ使えます。

GAS連携 重複自動除外 期間指定取得
03 / 03
🌸
導入5分で完結 ITが苦手でも大丈夫

画面の指示に従って入力するだけのウィザード形式。 必要なソフトのインストールもボタン1つで自動完了。 別途サーバーも契約も不要。WindowsパソコンさえあればすぐOKです。

Windows対応 ウィザード形式 サーバー不要
🔒
プラットフォームに迷惑をかけない設計です

Chatlady Manager は「人間が手動でやっていた操作をプログラムで代行する」ツールです。 サイトのシステムに直接アクセスしたり、データを書き換えたりすることはありません。 通常のブラウザ操作と同じ動作をするため、各プラットフォームへの影響はなく、安心してご利用いただけます。

// How It Works

導入は3ステップ

初回セットアップが完了したら、あとはツールを好きなときに起動するだけです。

01

install.bat をダブルクリックする

必要なソフト（Python・Playwright・Chromium）を自動インストール。すでに入っていれば自動でスキップします。

02

初回セットアップを完了する

ウィザードに沿って事務所ID・GAS URL・各サイトのID/PWを入力するだけ。Googleスプレッドシート+GASの準備は取扱説明書に沿って約10分で完了します。

03

「データ取得を開始」を押す

期間を選んでボタンを1回押すだけ。これだけで全サイトのデータ取得からスプレッドシートへの記録まで完了します。

👆
